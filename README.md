# pshr
An engine for Ruby on Rails to provide polymorphic uploads (using [shrine](https://github.com/shrinerb/shrine)) with file processing and an uploader interface.

## Installation
Add pshr to your Gemfile…

```ruby
gem 'pshr'
```

… and bundle install.
```bash
$ bundle
```

Or install yourself as:
```bash
$ gem install pshr
```
